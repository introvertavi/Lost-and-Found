# Lost and Found

A community-driven platform to help people locate and report lost or found items.

## Features

- **Report Lost Items**: Create detailed listings for items you've lost
- **Report Found Items**: Share information about items you've found
- **Search Database**: Browse through lost and found items in your area
- **Contact System**: Connect with other users to arrange item returns
- **Item Categories**: Organized listings by category (electronics, documents, accessories, etc.)
- **Location Tracking**: Specify where items were lost or found
- **User Profiles**: Build a reputation within the community

## Installation

1. Clone the repository:
```bash
git clone https://github.com/introvertavi/Lost-and-Found.git
```

2. Navigate to the project directory:
```bash
cd Lost-and-Found
```

3. Install dependencies:
```bash
npm install
```

4. Start the application:
```bash
npm start
```

## Usage

1. **Report an Item**: Click "Report Lost" or "Report Found" and fill in the details
2. **Search Items**: Use the search bar to find items by name, category, or location
3. **Contact Others**: Reach out through the built-in messaging system
4. **Verify Claims**: Confirm item ownership through security questions
